# App Review Notes Draft

Review Notes:

ReadOutLoud is a macOS app that converts English text into spoken audio files.

Important behavior for review:

1. The app requires an active internet connection because it uses Microsoft Edge TTS to generate speech.
2. Users can paste text directly or import a local `.txt` file.
3. Users can choose an output folder and save generated `.mp3` files locally.
4. The app does not require account creation or login.
5. The app now includes in-app Support and Privacy Policy entry points.

If audio generation fails during review, please verify that outbound network access is available. The app cannot generate speech while offline because the TTS service is cloud-based.
