When contacting support

  • Describe what you were trying to do.
  • Include the macOS version and app version if possible.
  • Attach a screenshot if the problem is about layout or export behavior.
  • Say whether the issue happens every time or only sometimes.

Common notes

  • ReadOutLoud requires internet access to generate speech.
  • The app can import local text files and export local mp3 files.
  • If audio generation fails while offline, reconnect and try again.
